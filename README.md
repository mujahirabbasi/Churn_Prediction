# Churn Prediction

This project explores employee churn data to find patterns and insights using machine learning techniques.  
The dataset contains information about 1,470 employees with 26 features.  

## Project Goal
The main goal is to analyze employee data, identify meaningful groups, and understand factors related to churn.

## Methods Used
- Data preprocessing
- Exploratory data analysis
- K-means clustering
- PCA
- t-SNE visualization

## Dataset
The project uses `Employees.csv`, which includes employee-related features such as education, job satisfaction, work-life balance, and performance rating. :contentReference[oaicite:1]{index=1}

## Outcome
The analysis helps reveal employee segments and provides insights that may support retention strategies.
